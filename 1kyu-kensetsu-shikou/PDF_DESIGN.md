## 印刷設定推奨値

### 一般設定
- **両面印刷**: 推奨
- **カラー**: モノクロ（白黒）
- **用紙**: 普通紙またはコピー用紙
- **製本**: 左綴じ

### フォントサイズ
- **大見出し（年次）**: 18pt ゴシック体 太字
- **中見出し（問題番号）**: 14pt ゴシック体 太字
- **本文**: 10.5pt 明朝体
- **ルビ**: 6pt
- **図**: 9pt Courier

## 生成方法

### 方法1: LaTeXを使用
```bash
# LaTeX形式で生成
pdflatex textbook.tex

# または
lualatex textbook.tex  # ルビ対応
```

### 方法2: Markdownから変換
```bash
# Pandocを使用
pandoc textbook.md -o textbook.pdf \
  --pdf-engine=lualatex \
  -V documentclass=ltjsarticle \
  -V geometry:margin=20mm
```

### 方法3: HTMLから印刷
```
1. index.htmlをブラウザで開く
2. 印刷機能（Ctrl+P）を使用
3. 「PDFに保存」を選択
4. 余白・レイアウトを調整して保存
```
