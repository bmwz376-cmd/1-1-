# 1級建築施工管理技士 第一次検定 完全オリジナル問題集

全250問の完全オリジナル問題集です。

---

## ✨ 特徴

- **完全オリジナル**: 実在する過去問題を一切再現していません
- **全250問**: 5年次×50問の体系的な構成
- **詳細解説**: すべての問題に解説と図解付き
- **ルビ振り**: 専門用語に読み仮名を完備
- **複数形式**: テキスト・JSON・HTML形式で提供

---

## 📚 構成

### 年次別構成
- **第1年次（基礎理解中心）**: 50問
- **第2年次（標準知識）**: 50問
- **第3年次（判断力重視）**: 50問
- **第4年次（実務応用）**: 50問
- **第5年次（総合応用）**: 50問

### 分野別構成（各年次共通）
- **法規**: 10問（建築基準法、建設業法、労働安全衛生法等）
- **施工（躯体・仕上）**: 15問
- **施工管理（工程・品質・安全・環境）**: 20問
- **共通（用語・材料・数量）**: 5問

## 🚀 使い方

### 方法1: テキストファイルで学習
```bash
cat textbook_all_250.txt
```

### 方法2: Webブラウザで学習
```bash
# ブラウザでindex.htmlを開く
open index.html  # macOS
xdg-open index.html  # Linux
start index.html  # Windows
```

### 方法3: JSONデータを活用
```python
import json

with open('all_250_questions.json', 'r', encoding='utf-8') as f:
    questions = json.load(f)

# 好きな形式で活用
for q in questions:
    print(q['question_html'])
```

## 🎯 学習の進め方

1. **第1年次から順に**: 基礎から応用へ段階的に学習
2. **分野別に集中**: 苦手分野を重点的に
3. **解説を熟読**: なぜそうなるかを理解
4. **図解で視覚化**: 概念を図でイメージ
5. **復習を繰り返す**: 間違えた問題は何度も
